# XStitchLabs Landing Page Design

## Purpose

Simple placeholder landing page for xstitchlabs.com — gives the QR code on packaging somewhere to point, establishes brand presence. Will later evolve into a proper showcase/shop.

## Technical Approach

- Single `index.html` with inline `<style>` in a new `/website` directory
- No frameworks, no build tools, no separate CSS files
- Hosted on Railway (user's existing account)
- Responsive (mobile-first — QR code scans will land here)

## Structure

### Header/Hero
- "XStitchLabs" brand name — clean sans-serif
- "Hanseatic Collection" series label
- Tagline: "Cross-stitch the cities of the Hansa"

### Featured Design
- Rendered stitch preview image of The Three Brothers (`example_image_3_brothers_riga.png` copied into `/website`)
- "No. 01 — The Three Brothers, Riga"
- Description: "The Three Brothers — a unique complex of medieval houses dating from the 15th and 16th centuries, the oldest residential buildings in Riga."
- Stats: 1,182 stitches · 8 colours · Medium difficulty
- Kit contents: 14-count Aida fabric, DMC stranded cotton, tapestry needle, pattern sheet

### Footer
- "More designs coming soon"
- Instagram link: @hanseatic.tomek
- xstitchlabs.com

## Style

- Clean, minimal — lots of whitespace, muted palette
- Typography: system sans-serif stack for body, UnifrakturCook (Google Fonts) for blackletter accent on design name — the medieval/Hanseatic nod
- Colour palette drawn from The Three Brothers design: sage green, warm beige, muted terracotta, coffee brown
- Responsive — works on mobile
