# Physical Cross-Stitch Kit Design

## Overview

Design for producing physical cross-stitch kits featuring Hanseatic architecture designs (Riga, Bergen, Lübeck). Initial test run of 12-15 kits for friends and family feedback.

## Target Audience

Mix of casual crafters and experienced stitchers. Kits should be polished enough for critical eyes, but don't need extensive hand-holding for beginners.

## Brand & Aesthetic

- Clean, minimal packaging (Muji/Aesop style)
- Quality materials, not over the top
- Design preview as the main visual element
- Typography-focused, no decorative clutter

---

## Kit Contents

### 1. Fabric
- **Type:** 14-count Aida cloth, white or natural
- **Size:** 4.5 inch (11cm) squares
- **Source:** Buy 1 meter bulk, cut to size
- **Cost:** ~£10-15 per meter (64+ squares)

### 2. Threads
- **Type:** DMC stranded cotton
- **Quantity:** ~6-8 colors per design, 2m per color per kit
- **Preparation:** Cut from skeins, wind onto small cards with DMC code labels
- **Source:** Individual DMC skeins (~£1 each)
- **Cost:** ~£25-40 for all designs

### 3. Needle
- **Type:** Tapestry needle, size 24 (blunt tip, large eye)
- **Source:** Bulk pack of 25
- **Packaging:** Pushed through small card square for protection
- **Cost:** ~£6-8 for bulk pack

### 4. Pattern Sheet
- **Format:** A4 folded to A5
- **Contents:**
  - Front: Design name + small preview
  - Inside spread: Full pattern grid + color legend with DMC codes
  - Back: Thread list + basic info
- **Paper:** 120-160gsm matte
- **Source:** Local print shop or quality home printing
- **Cost:** ~£5-10 for 15 sheets

### 5. Instruction Sheet
- **Format:** Single A5 or A6 card (same for all kits)
- **Contents:**
  - How to separate strands (2 from 6)
  - Threading the needle
  - Basic cross-stitch technique diagram
  - Starting and finishing (no knots)
- **Optional:** QR code to video tutorial
- **Cost:** ~£5-8 for 15-20 sheets

### 6. Packaging
- **Type:** Slim rigid card box, tuck-flap style
- **Dimensions:** ~12cm x 9cm x 1.5cm deep
- **Approach:** Plain white/kraft boxes + printed label wrap
- **Label:** Design preview on front, brand name, design name
- **Source:** Blank gift boxes + home/shop printed wraps
- **Cost:** ~£15-20 for boxes + labels

## What's NOT Included

- Hoop (keeps kit slim, testers can stitch in-hand or use own)
- Scissors (assume testers have access)
- Advanced instructions (keep it simple)

## Assembly Workflow

### One-Time Setup
1. Finalize 3 designs through pipeline, export pattern PDFs with DMC codes
2. Design pattern sheet template (A4 → A5 fold)
3. Design generic instruction sheet
4. Design box label/wrap template with preview slot
5. Order all materials

### Per-Kit Assembly (~5-10 mins each)
1. Cut fabric square (batch-cut all at once)
2. Cut thread lengths, wind onto cards, label with DMC code
3. Bundle threads with rubber band or paper wrap
4. Insert needle into card protector
5. Print and fold pattern sheet
6. Stack: fabric → threads → needle → pattern → instructions
7. Insert into box, apply label/wrap, close

### Efficiency Tip
Assembly-line approach: complete each step for all 15 kits before moving to next step.

## Sourcing Notes

### Fabric
- Zweigart, DMC, or generic brands
- Available: Amazon, Lakeside Needlecraft (UK), 123Stitch (US), local craft stores
- Buy 1-2 meters for buffer and testing

### Threads
- DMC stranded cotton (industry standard)
- Same sources as fabric
- Note which colors overlap between designs to reduce skein count

### Needles
- John James, DMC, or Prym brands (all equivalent quality)
- Bulk packs from craft stores or Amazon

### Boxes
- Search: "small gift boxes", "playing card tuck boxes", "rigid card boxes"
- Suppliers: Amazon, eBay, packaging specialists
- Alternative: DIY from heavy card stock using templates
- **Example:** Tiny Box Company - Small Luxury Matchbox Style Box
  https://www.tinyboxcompany.co.uk/small-luxury-matchbox-style-box

### Printing
- Local print shop for quality + heavier paper
- Home printer acceptable if good quality
- Online services (Mixam, Printed.com) have high minimums, not practical for 15 units
